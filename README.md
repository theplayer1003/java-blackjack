        - [ ] `Dealer` 는 점수가 16점을 넘지 못 했으면 반드시 카드를 한 장 뽑는다.
        - [ ] `Dealer` 는 점수가 17점 이상이면 카드를 뽑을 수 없다.
- [ ] 플레이어와 딜러의 턴이 모두 끝나면 점수를 비교하여 승패를 반환한다.

## 도메인 객체 사전 및 책임

### Card

* `Card` 카드
    * 무늬와 숫자,기호 를 가지는 불변 값 객체(VO).
    * 스스로 블랙잭 룰에 따른 점수를 계산할 책임을 가진다.
* `Deck` 표준 덱
    * 52장의 유일한 `Card` 객체를 초기화하고 셔플 및 분배 하는 일급 컬렉션.
    * 카드들에 대한 도메인 예외를 처리한다.
* `Hand` 손패
    * 게임 참가자들이 받은 카드 리스트를 캡슐화한 일급 컬렉션.
    * 카드를 저장하고, 자신의 총합 점수를 계산하는 책임을 갖는다.

### Participant

* `Participants` 게임 참여자 그룹
    * `Dealer` 와 `UserPlayer` 들을 포괄해 관리하는 일급 컬렉션.
    * 생성 시점에 참여자 구성에 대한 도메인 불변성을 검증한다.
    * 참여자에게 카드를 일고라 분배하거나, 각 참여자의 턴 흐름을 제어하는 책임을 갖는다.

* `Name` 참여자들의 이름, VO
    * ``

### Aggregate Root

* `BlackjackGame` 게임 컨트롤러
    * 블랙잭 도메인의 애그리거트 루트.
    * 전체 게임의 큰 흐름을 주관하며, 세부 로직은 하위 도메인 객체들에게 위임한다.
