# java-blackjack

## RDD, OutSide-In, TDD

- [ ] 게임에 참여할 사람을 이름을 통해 생성한다.
- [ ] 게임을 시작하면 딜러와 플레이어에게 카드를 2장씩 분배한다.
  - [x] 덱은 52장의 유일한 카드로 구성된다.
  - [x] 카드는 자신의 문양과 숫자 명칭을 반환할 수 있다.
  - [x] 카드는 블랙잭 규칙에 따른 점수를 계산할 수 있다.
  - [ ] 딜러와 플레이어는 덱으로부터 카드를 받을 수 있다.
- [ ] 플레이어는 보유한 카드의 점수 합이 21점을 넘지 않는 한 계속해서 카드를 뽑을 수 있다.
- [ ] 딜러는 처음에 받은 2장의 합계가 16점 이하면 반드시 1장을 추가로 뽑고, 17점 이상이면 뽑을 수 없다.
- [ ] 플레이어와 딜러의 턴이 모두 끝나면 점수를 비교하여 승패를 반환한다.
